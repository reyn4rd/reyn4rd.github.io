# Reynard static site

Versão estática pronta para GitHub Pages.

## Estrutura
- `index.html`
- `sobre/index.html`
- `contato/index.html`
- `blog/index.html`
- `dicas-marketing/*.html`
- `politica-de-privacidade/index.html`
- `termos-e-condicoes/index.html`
- `assets/css/style.css`
- `assets/img/*`
- `CNAME`
- `.nojekyll`

## Como publicar
1. Crie um repositório no GitHub.
2. Envie todos os arquivos para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch principal e a pasta `/(root)`.
6. Aguarde a publicação.

## Domínio próprio
O arquivo `CNAME` já está incluído com `reynard.com.br`.

## Observações
- Ajuste e-mail, telefone e WhatsApp na página de contato.
- Se quiser, você pode criar novas páginas duplicando um dos arquivos existentes.
- Esta base não depende de Jekyll, Astro ou qualquer build.
